# SxTools Roadmap

## Milestone 1 - Core Detection
- Detect installed applications
- Read config file
- Print environment status

## Milestone 2 - Installation Engine
- Install missing apps via winget
- Basic user prompts
- Install queue support

## Milestone 3 - State System
- Remember user choices
- Skip / delay prompts
- "Never ask again"

## Milestone 4 - PowerShell Modules
- Detect updates
- Update modules interactively
- Add auto-update mode

## Milestone 5 - Linux Support
- Add apt/dnf/pacman abstraction
- Shared config system

## Milestone 6 - CLI Tooling
- sx doctor
- sx install
- sx update
